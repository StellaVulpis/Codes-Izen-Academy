[메서드(Method)]
-메서드는 기능을 갖지만 독자적으로 실행될 수 없고 반드시 main에서 실행시켜야 한다.
-한가지 기능을 가짐
-모든 메서드는 ()가 존재
-다른말로는 function(함수)이라고 한다
----------------------------------------------------------------------------
-this는 현재 클래스에서 사용 (필드의 변수들에 사용한다. 메소드 변수와 구분지을 수 있다.)
class
{
  int num=0;
  void Method()
  {
     int num=10;
     this.num = num;
  }
}
-super는 상속받은(extends) 클래스에서 사용, 자식 클래스의 생성자(첫줄)에서 
부모 클래스의 생성자를 호출할 때 사용한다. 
(메소드, 생성자의 매개변수도 호출할 수 있다.)
(super.메서드() | super(매개변수) | super())
